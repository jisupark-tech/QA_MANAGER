# QA_MANAGER

게임 운영/CS 업무를 AI로 자동화하기 위한 프로젝트입니다.

## 목적
- 반복적인 CS/운영 업무를 AI 에이전트로 대체하여 효율화
- Claude Code 기반 맞춤형 CS AI 에이전트 설계 및 구축
- Moltbook 등 노코드 도구와의 하이브리드 운영 전략 수립

## 구조
```
QA_MANAGER/
└── QA_AI/          # CS/운영 AI 기획 및 설계 문서
```

## 핵심 전략
1. **즉시 도입** — Moltbook으로 단순 FAQ 자동화
2. **맞춤 구축** — Claude Code 기반 CS AI 에이전트 개발
3. **장기 운영** — AI 주력 + 사람은 에스컬레이션만 담당
